## Overview

**Adaptive Quiz** is an Android application developed using **Flutter** that provides a personalized quiz experience by dynamically adjusting question difficulty according to the user's performance.

The system evaluates each response and determines the next question based on the learner's progress. Correct answers lead to more challenging questions, while incorrect responses trigger slightly easier ones. This approach helps maintain an appropriate difficulty level and encourages continuous learning.

The application is designed with a clean and simple interface to minimize distractions and allow users to focus on answering questions efficiently.

## Features

- **Adaptive Question Difficulty** based on user performance  
- **Real-time Response Evaluation**  
- **Android Application built with Flutter**  
- **Personalized Learning Experience**
